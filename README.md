# CPP-Grader

CPP-Grader is a grader for C++ language written in C++.

## Installation

Download the file grader.cpp

Then put the grader file, code file, and testcase in the same directory.

## Usage

Compile and run grader using this command.
```bash
g++ grader.cpp && ./a.out
```
